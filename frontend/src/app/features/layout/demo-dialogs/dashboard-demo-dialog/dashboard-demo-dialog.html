Score du bleu : (-15 ici), l'idée est que le bleu aurait un score qui, démarrant de 0, descendrait de jour en jour et aurait un seul limite ex : 45 (aller bouffer bleus. pro., aller en recall ou autre).

Pour éviter de descendre à -45 le bleu devrait faire des exercices de folklore qui lui feraient gagner des points (ex 4 questions correctes = +1point).

=> L'idée est de le forcer à étudier régulièrement son folklore

Events : Ils ne seront certainement pas visibles pour les bleus mais serviront aux parrains/ass/capes/anciens.
